## ğŸš€ Installation

2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for **"TadaTheme Universal"**
4. Click **Install**

### From VSIX

## ğŸ¯ Usage

### Select a Theme

1. Open command palette: 
   - Windows/Linux: `Ctrl+Shift+P`
   - Mac: `Cmd+Shift+P`
2. Search for **"TadaTheme: Seleccionar Tema"**
3. Choose your favorite theme from the list
4. Done! Your editor will change instantly

### Restore to Original

1. Open command palette: `Ctrl+Shift+P`
2. Search for **"TadaTheme: Restaurar Original"**
3. Your VS Code will return to its default appearance
